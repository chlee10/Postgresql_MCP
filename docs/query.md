## 💡 쿼리 작성 팁

### 차트 요청 키워드
```
원그래프, 파이차트 → 🥧 파이 차트
막대그래프, 바차트 → 📊 막대 차트
라인차트, 추이, 추세 → 📈 라인 차트
```

### 효과적인 쿼리 작성법
```
✅ 좋은 예시:
"부서별 인원수 Top 10을 원그래프로 그려줘"
"경영관리 부서 직원 중 근속 5년 이상 명단"

❌ 피해야 할 예시:
"데이터 보여줘" (너무 모호함)
"전체 다 조회해줘" (결과가 너무 많음)
```

### 자주 사용되는 필터
- **부서**: 경영관리, 영업기획, IT기획, 재무 등
- **직급**: 사원, 대리, 과장, 차장, 부장, 팀장
- **기간**: 최근 N년, 올해, 지난달
- **조건**: N년 이상, Top N, N명 이상
